### 1. Install Docker Desktop
1. Restart your computer if required by the installer.
2. Launch the **Docker Desktop** application and keep it running in the background.

### 2. Download or Clone the Code
```git clone https://github.com/Irron21/ninongs-grab```
 ```cd ninongs-taxi```
### 3. Start the Multi-Container Environment:
Open your command prompt/terminal inside the root project directory and run:
```docker compose up --build -d```

### 4. Access the Interface:
Once the orchestration successfully finishes, launch your web browser and navigate to:
http://localhost
